# Squads-Recetas-con-IA
Buenas, este proyecto fue realizado con JS vanilla sin ningún framework con la utilización de Node JS y Vite, trata de una página que replica una serie de alimentos que puedes tener a disposición y con ellos una IA da una respuesta de posibles recetas a realizar.
En la descarga del repositorio hay un video ejemplo de como funciona la página y para la utilización suya unicamente ha de descargar las librerias correspondientes de Node para ejecutar en local.
La API KEY utilizada para las respuestas de la IA están en otro archivo aparte que cree y por motivos de seguridad no subí.
